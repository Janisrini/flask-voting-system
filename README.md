# flask-voting-system
A simple online voting system built with Flask and MySQL. Users can vote for candidates, view results, and manage candidates.
Project Name:
Flask Voting System

Description:
A simple online voting system built with Flask and MySQL. Users can vote for candidates, view live results, and manage candidates.

Features:

Cast votes securely with session tracking to prevent multiple votes

View real-time voting results in a table

Add or remove candidates using the admin interface

Simple and responsive frontend with HTML, CSS, and JavaScript

Technology Stack:

Python 3

Flask

MySQL

HTML / CSS / JavaScript

Future Enhancements:

Add user authentication for secure voting

Display voting results with charts and graphs

Deploy the system online for real-time access

Add email notifications for voters or admins

Installation / Setup:

Clone the repository

Install required Python packages (pip install -r requirements.txt)

Create the MySQL database and tables

Run the Flask app (python app.py)

Open http://127.0.0.1:5000/ in your browser
